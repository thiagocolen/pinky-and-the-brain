Welcome to the official documentation site for the **Pinky and the Brain Agents Service** - a production-ready, cloud-native agent service built with Deep Agents on LangGraph.js and exposed through the Agent Client Protocol (ACP), a REST/SSE API, and the Model Context Protocol (MCP).

<!-- truncate -->

This site covers everything you need to run, use, and extend the service:

- **End-User Documentation** - installation, configuration, running the REPL/REST/MCP/ACP entrypoints, a command reference, and an FAQ / troubleshooting knowledge base.
- **Developer & Technical Documentation** - project structure, architecture, a module-by-module source code reference, and contribution guidelines.

Head over to the [Introduction](/docs/intro) to get started, or jump straight to [Installation](/docs/end-user/installation).
